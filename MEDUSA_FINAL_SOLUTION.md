## Deployment Instructions

### 1. Deploy Frontend to Vercel (10 minutes)

```bash
# Push to GitHub
git add .
git commit -m "Complete e-commerce frontend with Payload CMS"
git push origin main

# On Vercel:
1. Go to vercel.com
2. Import your GitHub repo
3. Framework: Next.js
4. Root Directory: frontend
5. Deploy
```

### 2. Deploy Payload CMS to Vercel (10 minutes)

```bash
# On Vercel:
1. Import same repo
2. Framework: Other
3. Root Directory: cms/payload
4. Build Command: npm run build
5. Output Directory: dist
6. Add Environment Variables:
   - MONGODB_URI=your_mongodb_atlas_url
   - PAYLOAD_SECRET=your_secret
7. Deploy
```

### 3. Update Frontend to Use Deployed CMS

```typescript
// frontend/.env.production
NEXT_PUBLIC_PAYLOAD_URL=https://your-cms.vercel.app
```
