## 📋 Submission Requirements

### Required Links

1. **GitHub Repository URL**
   - Format: https://github.com/username/repo-name
   - [ ] Repository is public
   - [ ] Contains all code

2. **Frontend URL (Vercel)**
   - Format: https://your-app.vercel.app
   - [ ] Homepage loads
   - [ ] All pages accessible

3. **CMS Admin URL (Vercel)**
   - Format: https://your-cms.vercel.app/admin
   - [ ] Admin panel accessible
   - [ ] Can log in

4. **Medusa Backend URL (Render)**
   - Format: https://your-backend.onrender.com
   - [ ] API responding
   - [ ] Admin accessible at /app

5. **Loom Video URL**
   - Format: https://www.loom.com/share/...
   - [ ] Video is public
   - [ ] Link works
